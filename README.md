# 📚 Online CBT Examination Portal

A fully functional **Computer-Based Test (CBT) system** built using HTML, CSS, and JavaScript.  
This project simulates a real government exam environment with timer, navigation, result analysis, and answer review.

---

## 🚀 Live Demo
👉 https://erpiyush2050.github.io/

---

## 🎯 Features

### 🧠 Exam System
- 150 Multiple Choice Questions
- Next / Previous navigation
- Question palette (jump to any question)
- Save answers automatically
- Clear answer option

---

### ⏱ Timer
- Countdown timer (1 hour)
- Warning popup at last 5 minutes
- Auto-submit when time ends

---

### 📊 Result Dashboard
- Total Correct, Wrong, Unattempted
- Percentage & Accuracy
- Grade (A / B / C / Fail)
- Visual charts (Pie chart)

---

### 📖 Answer Review
- All questions displayed after submission
- ✔ Correct answers highlighted in green
- ✘ Wrong answers highlighted in red
- ⚠ Unattempted questions in yellow

---

### 🧩 Section-wise Analysis
- Writer Section (Q1–Q50)
- Calculator Section (Q51–Q120)
- Mixed Section (Q121–Q150)

---

### 📑 Extra Features
- Download/Print result as PDF
- Clean and responsive UI
- Government CBT-style interface
